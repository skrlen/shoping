# shoping
## 这是一个开源的商城系统 
### 是为了对商品进行统一管理出售
### 由于前台设计时缺少部分界面，目前只支持到购物车系统
### 目前该项目已进入停止状态，完成时间未知！

目前项目已经开源，有兴趣的可以自行下载，使用Django版本为2.2，数据库为sqlite，python版本3.7.2

项目目前处部分功能以外，可以部署使用，新手可以借助该项目在云服务器/unix/linux进行部署练习
